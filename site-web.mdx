Esta página reúne as atualizações do site da Meu Ingresso voltadas ao product **Organizador**.

Sempre que uma nova versão for publicada, adicione um novo accordion no topo da lista com o número da versão, a data de publicação e um resumo claro das novidades entregues.

<AccordionGroup>
  <Accordion title="Versão 1.0.0" defaultOpen icon="sparkles">
    **Data de publicação:** DD/MM/AAAA

    ### Novos recursos

    - Descreva aqui as funcionalidades inéditas adicionadas ao site.
    - Informe o impacto prático para o organizador sempre que possível.

    ### Melhorias

    - Liste os aprimoramentos de usabilidade, desempenho ou fluxo.
    - Destaque ajustes que facilitem a operação no dia a dia.

    ### Correções

    - Registre bugs corrigidos e comportamentos ajustados.
    - Informe correções relevantes para navegação, formulários ou painéis.

    ### Observações

    - Adicione detalhes complementares, dependências ou limitações conhecidas.
  </Accordion>
</AccordionGroup>

## Como manter esta página atualizada

1. Crie um novo accordion no topo para cada nova versão publicada.
2. Use o título no formato `Versão X.Y.Z`.
3. Informe a data de publicação logo no início do conteúdo.
4. Organize as informações em `Novos recursos`, `Melhorias`, `Correções` e `Observações`.
5. Mantenha as versões mais recentes sempre acima das anteriores.
